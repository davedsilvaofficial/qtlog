# MASTER WBS FORMAT (Canonical)

WBS-X.Y.Z
• X = Program / Capability Domain
• Y = Artifact Class (Document Type)
• Z = Unique Artifact Identifier

SUB-WBS FORMAT (Optional Extensions):
- WBS-X.Y.Z.vN → version number
- WBS-X.Y.Z.aN → appendix / addendum
- WBS-X.Y.Z.eN → evidence item / exhibit

VERSIONING RULES:
• Meaningful change → increment vN
• Supporting attachment → aN or eN
• Never reuse a WBS code for different content
