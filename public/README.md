# qtlog (Public Overview)

This is a screened, public-safe overview of `qtlog`. It intentionally omits private implementation details.

## What it does
- Creates local, timestamped log entries
- Supports device tagging
- Can show an authoritative America/Toronto timestamp helper

## Non-goals (intentionally excluded)
- Any private keys, tokens, env files, or secrets
- Any logs or personal operational data
- Any internal repo paths, usernames, or device identifiers

## Example (sanitized)
STAMP_NOW: 2025-01-01 1200 EST
