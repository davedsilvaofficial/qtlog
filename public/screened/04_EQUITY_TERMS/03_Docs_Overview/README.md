This folder provides an overview of the documents used in connection
with the offering process.

Purpose:
- Explain the document flow to investors and advisors
- Align expectations prior to qualification
- Avoid publishing offering documents in the screened data room

Document flow (typical):
1. Investor qualification (accredited / professional)
2. Delivery of Private Placement Memorandum (PPM)
3. Subscription Agreement execution
4. KYC / AML compliance
5. Wiring instructions via escrow or counsel

Notes:
- Actual offering documents are provided only to qualified investors
- Distribution is managed via counsel and controlled channels
- Nothing in this folder constitutes an offer or solicitation
