## Disclosure Notes

- This data room emphasizes evidence over disclosure.
- Sensitive implementation details, source code, and classified
  materials are intentionally excluded.
- Certain materials may be provided only under NDA or to qualified
  investors.

## Versioning

All documents are date-stamped. Where multiple versions exist, the
most recent version should be considered authoritative unless noted.
