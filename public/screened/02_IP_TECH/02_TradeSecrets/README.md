# Trade Secrets (Primary IP Portfolio)

This folder documents proprietary trade-secret IP assets that are
intentionally not publicly disclosed via patents.

Contents should demonstrate:
- Existence of the IP asset
- Ownership and control
- Measures taken to maintain secrecy
- Date of creation and evolution (versioning)

Suggested contents:
- TradeSecret_Register_<YYYY-MM-DD>.md
- TradeSecret_Summary_<AssetName>_<YYYY-MM-DD>.md
- Access_Control_Notes_<YYYY-MM-DD>.md
- IP_Chain_of_Title_<YYYY-MM-DD>.md

Important:
Do NOT include source code, formulas, or sensitive implementation details.
This folder is for diligence evidence, not disclosure.
