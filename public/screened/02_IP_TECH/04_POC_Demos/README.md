# Proof of Concept (POC) / Demonstrations

This folder contains evidence that the technology has been demonstrated
in controlled, test, or pilot environments.

Purpose:
- Substantiate performance and feasibility claims
- Provide confidence without exposing trade secrets

Acceptable materials:
- Demo videos or screenshots
- Performance summaries and benchmark results
- Pilot or test reports
- Customer or partner validation statements

Suggested filenames:
- POC_Summary_<Project>_<YYYY-MM-DD>.pdf
- Demo_Video_<Project>_<YYYY-MM-DD>.mp4
- Performance_Results_<Project>_<YYYY-MM-DD>.pdf
- Pilot_Report_<Partner>_<YYYY-MM-DD>.pdf
